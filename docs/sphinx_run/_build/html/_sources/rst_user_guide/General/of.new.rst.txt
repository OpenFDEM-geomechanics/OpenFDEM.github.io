**of.**\ :red:`new`
    **(Recommend)** Clear all old memories from the last run and start a new run. 
    It is used as the first line in the input file. It also prevents the conflict 
    when two programs run at the same time.
    

**Example:**

.. code-block:: 

    # First line of the program
    of.new

**of.**\ :red:`finalize`
    **(Mandatory)** This command terminates the console at the end of the run 
    and clear all the old memory from the last run.

|

**Example:**

.. code-block:: 
    
    # '''...Code about the model...'''
    # Last line of the code
    of.final
